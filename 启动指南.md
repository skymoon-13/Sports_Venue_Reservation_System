# 系统启动和使用指南

## 第一步：启动应用

在项目目录下，确保虚拟环境已激活，然后运行：

```bash
python app.py
```

看到以下信息说明启动成功：
```
 * Running on http://0.0.0.0:5000
 * Debug mode: on
```

## 第二步：访问系统

打开浏览器，访问：
```
http://localhost:5000
```

## 第三步：登录系统

### 默认管理员账号
- **用户名**: `admin`
- **密码**: `admin123`

⚠️ **重要**: 登录后请立即修改密码！

## 系统功能说明

### 管理员功能

登录后，你可以：

1. **场馆管理**
   - 创建新场馆
   - 编辑场馆信息
   - 删除场馆
   - 设置场馆开放时间和收费标准

2. **预约管理**
   - 查看所有用户的预约
   - 修改预约信息
   - 取消预约

3. **数据统计**
   - 查看各场馆预约次数
   - 查看今日预约数
   - 查看总预约数

### 用户功能

注册普通用户账号后，可以：

1. **查看场馆**
   - 浏览所有可用场馆
   - 查看场馆详情（位置、容量、开放时间、收费标准）

2. **创建预约**
   - 选择场馆
   - 选择日期和时间段
   - 系统自动计算费用（学生收费，教职工免费）

3. **管理预约**
   - 查看我的预约列表
   - 取消预约（开场前1小时内禁止取消）

## 常见问题

### Q: 启动时提示端口被占用？

**解决方案：**
修改 `app.py` 最后一行，改为其他端口：
```python
app.run(debug=True, host='0.0.0.0', port=5001)
```

### Q: 访问页面显示 404？

**解决方案：**
- 确认应用已启动
- 确认访问的是 `http://localhost:5000`
- 检查浏览器控制台是否有错误

### Q: 登录后看不到管理员功能？

**解决方案：**
- 确认使用的是管理员账号（admin/admin123）
- 检查用户角色是否为 `admin`
- 刷新页面

### Q: 邮件功能不工作？

**解决方案：**
- 邮件功能需要配置 `config.py` 中的邮件服务器
- 如果不需要邮件功能，可以暂时忽略
- 通知会保存在数据库中，可以通过通知接口查看
